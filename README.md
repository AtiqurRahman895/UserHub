## How to Run the Project

1. Clone the repository:
```bash
git clone https://github.com/AtiqurRahman895/UserHub.git
cd UserHub
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. To build for production:
```bash
npm run build
```
